# Margin Degradation

> Expanding before fixing margins creates compounding problems

## Framework
- Open location 2: founder splits, margins drop on both
- Open location 3: founder further split, location 2 never recovered to location 1 margins
- Never fully utilized across 3 facilities because founder wasn't present

## How-To / Method
- Get location 1-3 to 35-40% margins first (requires founder exit from ops)
- Then open location 4 from proven margin model
- This prevents repeating margin decline pattern
