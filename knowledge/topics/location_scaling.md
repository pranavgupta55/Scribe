# Location Scaling

> Only add locations once existing model is proven to self-propagate

## Key Takeaway
- Validate word-of-mouth model at single location before replication
- This applies to all brick-and-mortar businesses

## Connections
- Add locations only after model self-propagates; scaling 1 to 5+ requires owner business time and can't execute if founder is 80% clinical. _(connects to: Multiple Location Expansion)_
- Can't scale locations if founder is required for training; systematize before expanding. _(connects to: Team Capacity Constraints)_
