# Resilience & Trauma

> Resilience = speed of return to baseline after disruption; trauma = permanent behavior change from bad event.

## Framework
- Resilient = bounces back quickly
- Not resilient = stays disrupted long
- Trauma = never returns to baseline (permanent shift)

## Examples
- Baby touches hot stove, never touches again = trauma (permanent avoidance)
- But was the trauma bad? It prevented harm; context matters
