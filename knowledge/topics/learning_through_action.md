# Learning Through Action

> Implementation is everything; reading without doing teaches nothing; implementation changes life.

## Framework
- Read 14 self-help books in quarter = zero life change
- Vow to implement everything in next book before next = quit job, drove across country, things changed
- What you do next matters more than what you did last

## Key Takeaway
- Can change entire bloodline moment you realize next action matters most
- Implementation gap is where most people fail
