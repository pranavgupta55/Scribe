# Providing Value

> Value = helping others get what they want

## Framework
- Zig Ziglar: 'You can get whatever you want in life as long as you help enough other people get what they want'
- Step 1: Do epic stuff (gain experience, build skills)
- Step 2: Talk about it (share the insights you gained)
