# Value Stacking

> Separate one-time skill-building (high-value) from consumables (low-value); price accordingly

## Framework
- One-time: learning to close (high value, one-off skill)
- Consumables: leads, ongoing support (lower value, repeating need)

## Key Numbers
- One-time skills: 5–10x higher price than consumables

## Why It Works
- Removes churn: once advisors learn skill, consumable value is obvious and sticky
- Prevents conflating frontend value with continuity value

## Claims
