# Knowledge Types

> Declarative (knowing about) and procedural (knowing how) knowledge are different; watching videos doesn't replace doing

## Framework
- Declarative: memorizing facts, learning how private equity works
- Procedural: actually doing cold calls, doing deals, executing

## Key Takeaway
- 100 cold calls teach more than 100 books/videos
- First 100 are hard because you have to learn procedurally, not declaratively

## Implications
- Can't shortcut execution by learning declaratively
- Must do the thing to learn how to do the thing
