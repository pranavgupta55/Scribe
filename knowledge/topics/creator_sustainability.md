# Creator Sustainability

> Audiences need reminders more than new ideas. You don't need novel content each day; you need different angles on same themes.

## Framework
- Dave Ramsey: 35 years of 'stop spending money'
- Each new clip novel context for same message
- $22M lottery winner vs. $2M lottery winner—answer similar, but framing differs
