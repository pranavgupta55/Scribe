# Effective Visuals vs. Razzle

> Visuals should clarify, not distract - use data charts and clear text, not flame effects.

## Framework
- Effective: on-screen text, data visualizations, charts showing trends
- Distracting: changing backgrounds, flame effects, pop-ups, transitions

## Examples
- Changing background color behind speaker: adds 0 educational value
- Chart showing revenue growth over time: adds clarity and scale
