# Book Series Architecture

> One book split into three (Offers → Leads → Money Models) to stack skills progressively; each layer adds prerequisite knowledge.

## Framework
- Original: one massive book with all three topics
- Revised: three sequential books where each builds on prior
- Money Models book released last because it completes the sequence

## Key Takeaway
- Releasing Money Models alone would confuse readers ('I don't know how to do Offers or Leads')
- Sequential release positions full system in customer mind
