# Skill Stacking

> Layer complementary skills to compound value; each skill multiplies leverage of previous skills

## Framework
- Jay-Z: rhythm → rap → lyrics → sales → promotion → artist recruitment → label
- Finance: math → bookkeeping → accounting → taxes → insurance → M&A

## Why It Works
- Each skill requires previous layers; builds on prior knowledge
- Combined skills worth exponentially more than single skill
- Selling multiplies all skills: same product with sales >> product without sales

## Key Takeaway
- Winners win by continuously layering skills; no skill is wasted because it compounds with others

## Common Mistakes
- Waiting for perfect first skill before moving to second
- Not realizing each new skill multiplies leverage of all previous skills
