# Content and Creative Strategy

> Video underperforms because execution quality varies; static images from camera roll + organic daily posting + repurposed podcast clips maximize ROI.

## Video vs. Image Volatility
- Video: highest upside (best ads are video) but highest downside (worst ads are video)
- Images: more consistent; customer imagination fills in compelling narrative better than poor video
- Solution if video fails: Couplepreneurs make great content (Instagram reels on masculinity, relationships) but didn't run as ads

## Organic Daily + Ad Repurposing
- Post 1 piece of organic content daily minimum (currently 2/week); use camera roll, behind-scenes, clips from life
- Take top-performing organic pieces, append 5-sec CTA, run as paid ads
- Cost: free testing ground; identify winners before spending ad budget on unknowns

## Static Image Scale
- Run 100+ static images from iPhone camera roll; currently running 12-16
- Images should be personal moments, couple activities, event footage, testimonials
- Copy stays constant once proven; only creative (image/video hooks) changes

## Podcast Repurposing
- Slice best moments from podcast interviews into 30-60 sec clips
- Run clips as both organic posts AND ads with CTA appended
- Podcast interview that converted hosts into clients = untapped goldmine for content

## Ad Copy Strategy
- Once copy works, never change it; only swap creative (hooks/angles)
- Example: ran same copy for Acquisition.com for an entire year while testing 100s of creative variations
- Simplicity in messaging beats feature lists; focus on one core benefit per audience segment
