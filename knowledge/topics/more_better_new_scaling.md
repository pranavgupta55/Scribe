# More, Better, New Scaling

> Three-step framework for increasing lead volume: more volume, improved efficiency, then new platforms/methods

## Framework
- More: 100 primary actions per day for 100 days (inputs guarantee outputs)
- Better: Test one variable per week per platform; prioritize constraint (step with biggest drop-off)
- New: New placements → new platforms → new Core Four activity

## Key Numbers
- 5% improvement on constraint = 2x improvement in leads; 5% improvement elsewhere = 1.1-1.16x improvement
- For businesses under $1M profit: do 'more' first; over $1M: flip to 'better' then 'more'

## How-To / Method
- Rule of 100: commit to one platform, 100 primary actions per day, for 100 days minimum
- Test Monday each week; look at results next Monday with three-step process: pick winner, log results, create next test
- If can't beat current best in four weeks, move to next constraint
