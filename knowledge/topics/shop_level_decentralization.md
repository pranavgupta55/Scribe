# Shop-Level Decentralization

> Every location needs a dedicated ops manager to run independently; separate P&L enables accountability without bottom-line tie.

## Framework
- Run P&L separately per shop (or LLC) so shop ops managers have visibility
- Ops manager has no tie to shop P&L but is accountable for operations
- Enables true decentralization; removes need for owner hand-holding
