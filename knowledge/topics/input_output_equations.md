# Input-Output Equations

> Define the specific action that drives results, then multiply it

## Framework
- Step 1: Define what action gets you closer (input-output equation)
- Step 2: 10x that action
- Step 3: 100x that action if possible

## Common Mistakes
- Skipping the definition step
- Working blind toward goal
