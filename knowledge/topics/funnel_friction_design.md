# Funnel Friction Design

> Add layers of friction before consultation call to reduce trash calls

## Framework
- Currently: Opt-in PDF → Everyone sees scheduler → 2 reps take all calls
- Problem: No filtering between PDF and consultation call

## Solution
- Layer 1: Change ad copy to reinforce 65+ targeting
- Layer 2: Add wound size question to application
- Layer 3: Score and filter before showing scheduler
