> Top performers put in outsized work that compounds into exponential rewards

## Key Takeaway
- 100x more effort can yield 1M times payoff
- From relative perspective, exponential returns from being best

## Framework
- Absolute perspective: diminishing returns
- Relative perspective: exponential returns from excellence
