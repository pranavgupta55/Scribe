# Learning Strategy

> All learning is additive; winners extract lessons from failures and improve behavior

## Key Takeaway
- Learning never hurts; skills are inflation-proof
- Bad experiences teach what not to do; still serve you if you change behavior

## Framework
- Winners frame everything as serving them (learning opportunity), not them serving it (setback)

## Claims
- Skills are inflation-proof; learning never hurts; winners succeed by learning from failures (learning what not to do) — [how_to_catch_up_in_life_using_logic_EonibwnAEME.txt]

## Connections
- Learning is behavioral change measured by exposure count; winners extract lessons from failures. _(connects to: Learning Definition)_
- Winners compound faster by extracting lessons from failures and focusing on controllable variables, not initial conditions. _(connects to: Speed Of Learning Over Starting Point)_
- Learning compounds through extracting lessons from failures and deliberately improving behavior in repeated conditions. _(connects to: Learning Behavior Change)_
- Winners extract lessons from observing differences between high and low performers through scientific detachment. _(connects to: Observational Learning)_
- Both emphasize that true learning requires behavior change through iteration, not passive consumption. _(connects to: Learning Through Iteration)_
