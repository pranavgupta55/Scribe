# Product-Market Goodwill

> Gap between price paid and value received; building it drives referrals and repeat purchases

## Key Takeaway
- Higher goodwill = more referrals; best customers are those who exceed expectations

## How-To / Method
- Sell better (higher-quality) customers; they get more value and refer more
- Set lower expectations via conservative promises; room to overdeliver
- Deliver wins fast and often; 48-hour first impression window critical
- Remove friction; add progress updates, set next meeting always, never leave customer in limbo
- Iterate product monthly using customer feedback; solve biggest common problem each cycle
- Upsell new offerings continually; more purchases = more opportunities for value

## Implications
- Customer goodwill = business insurance; makes everything else easier

## Claims
- Customer goodwill drives referrals; build it by selling better customers, setting lower expectations, ensuring faster wins, improving product, and continuously upselling new offerings — [100m_leads.txt]
