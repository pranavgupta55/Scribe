# Process Mapping

> Visualize your entire business as a linear workflow from idea to delivery.

## How-To / Method
- Draw your business from end-to-end; if you can't, how do you have a business?
- Start with big-picture stages (idea → script → shoot → edit → post)
- Subdivide each stage until you can't reduce further (e.g., post = check schedule, add 3 hours, select time slot, compose caption, upload, schedule)

## Key Takeaway
- One high-level step often masks 6-7 granular actions
