# Hotline Workflow Example

> Real-world example: AI-driven transcription and clip editing workflow breaks into discrete automatable steps.

## Framework
- Step 1: Transcribe hotline call
- Step 2: Segment speakers (A vs B initially, then A vs C when new speaker joins)
- Step 3: Isolate individual conversations (10 separate clips)
- Step 4: Find highest tension moment in each clip
- Step 5: Remove fillers (ums, ahs)
- Step 6: Collapse non-essential data points
- Step 7: Export final file

## Key Takeaway
- Each step is its own prompt/agent, trainable separately, updatable like training a human
