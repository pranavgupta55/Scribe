# Directional Approach

> Ask customers qualitatively where they first heard about you for simple allocation signal

## Framework
- No complex tech required
- Customer memory strongest on first touchpoint
- Provides direction for reinvestment without perfect data

## Examples
- 2/3 found via YouTube first
- 1/3 told about YouTube by someone else
- Smaller portions via Instagram, book, other channels
- YouTube showed clear priority
