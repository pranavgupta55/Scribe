# Audience Targeting

> Narrowing cold audience to highest-probability buyers via platform-native targeting tools

## How-To / Method
- Start with lookalike audience from best customer list
- Layer demographic filters: age, income, interests, location
- Local audiences already narrow (0.1% of nation); add few filters only

## Framework
- Efficiency trades for reach: narrow targets = higher response rate but faster burnout; broaden = lower response but more inventory

## Claims
- Use lookalike audiences based on best customers, then add demographic/interest filters; start specific to burn through smaller audiences profitably before expanding — [100m_leads.txt]
