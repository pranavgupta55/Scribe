# Location Economics Sequencing

> Make sure current location is maxed before opening next

## Framework
- First location: 500-700 pizzas per day
- Owner 33%, partner 67%
- Subsequent locations: 50-50, then 100%
- Incentivizes opening new locations

## Implications
- Owner benefits from new locations (100% of third+)
- Should max first location before splitting focus
