# Product Avatar Diversity

> Use AI avatars or multiple user demographics in ads; algorithm targets by who's in the image, not targeting settings.

## Framework
- Facebook algorithm reads faces in ads and targets similar demographics
- Create avatars of different ethnicities saying same value prop
- Each avatar reaches different saturation point in same broad audience

## Examples
- AI avatar: "I went around world for $1,200, here's objections and proof"
- Different genders, ages, ethnicities = multiple saturation curves

## Why It Works
- Likeness is strong predictor of conversion per algorithm
- Expands addressable audience without changing targeting parameters

- Facebook algorithm reads who's in ads; if owner is Asian lady, algorithm shows to Asian ladies because they're most likely to convert via likeness. — [building_a_12_000_000_business_for_a_stranger_in_25_minutes_MH-IMJxbUY4.txt]

## Connections
- Exclude low-probability avatars early; algorithm targets by image demographics, not settings. _(connects to: Avatar Targeting)_
