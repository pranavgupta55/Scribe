# Bonuses

> Additional items stacked to expand price-to-value discrepancy without changing core offer; tools and checklists more valuable than trainings.

## Strategy
- Present bonuses after core offer accepted in 1-on-1 selling
- Each bonus should address specific objection or next logical need
- Total bonus value should exceed core offer value
- Stack bonuses one-by-one for cumulative perception building

## Types
- Proprietary tools: calculators, templates, checklists (high value, low effort)
- Recorded content: webinars, workshops, tutorials (created once, given infinitely)
- Partner products: negotiate free/discounted goods from adjacent businesses
- Access: VIP status, exclusive events, higher service level for trial period

## How-To / Method
- Give special name with benefit: 'Foolproof Grocery System' vs 'Bonus #3'
- Articulate how it relates to their issue and improves their life
- Add scarcity: 'Only with this offer, never for sale elsewhere'
- Add urgency: 'If you buy today, I'll add this $1,000 bonus free'
- Ascribe price tag and justify it with specifics
