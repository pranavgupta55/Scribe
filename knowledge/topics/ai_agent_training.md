# AI Agent Training

> AI agents learn like toddlers: explicit, granular, observable steps, but never forget.

## How-To / Method
- Break a role into 16 actual activities
- Take one activity at a time
- Train an agent via prompt on that single activity
- Iterate and update the prompt (updating training)

## Common Mistakes
- Telling an agent nothing and expecting it to work = same mistake as with humans, but we believe machines should do it anyway

## Claims
- Teaching an AI agent requires the same specificity as teaching a toddler, down to granular actions, except the machine never forgets. — [why_ai_agents_will_replace_your_next_hire_rMf-JuikR-Q.txt]
