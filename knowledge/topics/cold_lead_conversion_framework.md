# Cold Lead Conversion Framework

> VSL → pre-frame → qualification → close; each step filters and prepares prospect for commitment

## Framework
- VSL: establish credibility and explain dream outcome
- Pre-frame call: set agenda, explain next steps
- Qualification: confirm problem/ICP match
- Close: commitment ask (booking, payment, etc.)
