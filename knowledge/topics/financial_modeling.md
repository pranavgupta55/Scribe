# Financial Modeling

> Run the math: 100 locations at 40% margins with 7% royalty = $36M exit; 20 locations at higher efficiency = $80M with less risk.

## Key Numbers
- 100-location target: $100M top line, 7% royalty = $7M annual take, but need to improve margins
- 20-location alt: $500K profit each × 20 = $10M EBITDA, 8x multiple = $80M

## Implications
- 20 locations is faster, simpler, lower operational overhead
- 100 locations requires more systems and franchisor support before franchisees will buy
