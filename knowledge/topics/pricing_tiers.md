# Pricing Tiers

> Only add lower tiers if ideal customers cite price; if non-ideal customers complain, price filter is working.

## Examples
- School added $9/month tier (11x cheaper) for hobby groups, kept $99 premium for high-touch

## Framework
- Ideal customers say price is problem → add lower tier
- Non-ideal customers say price is problem → ignore (it's a filter)
