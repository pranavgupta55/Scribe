# Learning Definition

> Learning is behavioral change in the same condition, measured by how many exposures it takes.

## Framework
- Same condition + new behavior = learning
- Same condition + same behavior = no learning

## Examples
- Knock-knock (condition) → new response (behavior change) = learning happened
