# Script Discipline

> Scripts must be breathed, lived, and enforced; no ad-libbing or cowboy behavior allowed

## Framework
- Everyone reads same script word-for-word
- Daily training to ensure consistency in delivery
- If no script adherence, remove person from team

## Common Mistakes
- Scripts treated as suggestions or guidelines, not law
- No consequences for going off script to close deals

## Implications
- Off-script salespeople make promises company can't deliver on
- Creates long-term liability and service failures
