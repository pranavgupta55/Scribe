# Scaling Discipline

> Perfect business model before scaling; ego drives premature scaling, which compounds broken economics.

## Common Mistake
- Entrepreneurs scale before nailing model because ego is tied to growth, not unit economics
- Scaling broken model = scaling problems across more locations/reps, harder to fix

## Better Path
- Spend 3 years perfecting model, get all metrics right, then scale fast
- Preferable to spending 3 years fixing broken model at scale

## Application
- Apply Rule of 100 to each new channel before scaling it
- Hire more salespeople before expanding location/offerings
- Say no to expansion until unit economics locked in
