# Observable Behavior Specification

> Machines require explicit, observable, measurable descriptions; vague language fails both humans and AI.

## Common Mistakes
- Vague: be more charismatic, be better, off-brand
- Clear: raise voice 2 octaves, talk 20% faster, nod when other person speaks, use 3-word sentences

## Caveats
- If you can't describe it in black-and-white language to a toddler, you can't train an agent on it
- This is harder than it sounds but is the actual work of AI implementation

## Claims
- Machines don't understand vague language (be more charismatic) and humans don't either; both require black-and-white specifications (raise voice this way, talk faster, nod when other speaks). — [why_ai_agents_will_replace_your_next_hire_rMf-JuikR-Q.txt]
