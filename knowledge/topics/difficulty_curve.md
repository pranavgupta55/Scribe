# Difficulty Curve

> All difficult things follow same curve: optimism → informed pessimism → despair → breakthrough → success

## Framework
- Stage 1 (Uninformed Optimism): "This will be easy, I've got this"
- Stage 2 (Informed Pessimism): "Oh, I see the downsides now"
- Stage 3 (Valley of Despair): "This is hard forever, I should quit"
- Stage 4 (Upside appears): "Wait, this is working"
- Stage 5 (Goal achieved): "Made it"

## Common Mistakes
- Quitting at Stage 3 and jumping to new thing's Stage 1
- Never getting through to Stage 4/5
- Thinking hard lasts forever instead of recognizing it will end
