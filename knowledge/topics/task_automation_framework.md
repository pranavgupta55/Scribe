# Task Automation Framework

> Decompose daily work into granular tasks, then iteratively automate starting with the first.

## How-To / Method
- List daily activities at granular level (respond to emails, write copy, edit, analyze)
- Don't chunk (e.g., "I run ads" has subtasks: make campaigns, set budgets, analyze, create, write, test)
- Put first task into AI; ask for automation steps
- Screenshot and ask "what do I do now" iteratively if stuck

## Key Takeaway
- Everyone has AI tutor at fingertips; most don't use it
- Granular decomposition unlocks rapid automation loops
