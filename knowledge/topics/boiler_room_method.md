# Boiler Room Method

> Five-stage daily rotation: Intro, Discovery, Offer, Objections/Looping, Hot-Topic

## Framework
- Stage 1: Intro (opening, rapport)
- Stage 2: Discovery (qualifying questions)
- Stage 3: Offer (present solution)
- Stage 4: Objections & Looping (handle pushback, loop back)
- Stage 5: Hot-Topic (current weak spot of the week)

## Why It Works
- Crisp script prevents quality collapse
- Daily practice embeds muscle memory
- Rotates focus so all stages stay sharp
