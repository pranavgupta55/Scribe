# Problem-Solution Framework

> Five-step process to create valuable offers: identify dream outcome, list all problems, turn into solutions, generate delivery vehicles, trim and stack.

## Five Steps
- Step 1: Identify Dream Outcome (what they want, specific destination)
- Step 2: List Problems (every obstacle preventing achievement, in detail)
- Step 3: Create Solutions (turn each problem into solution statement)
- Step 4: Delivery Vehicles (all possible ways to deliver each solution)
- Step 5: Trim & Stack (remove low-value, bundle remainder into offer)

## Key Takeaway
- Must solve every perceived problem or risk losing the sale
- One missing piece can be the sole reason someone declines
- 16-64 problems typical; comprehensive list creates 4,351+ value perception
- Each bonus should address specific concern in prospect's mind

## Delivery Cube Cheat Codes
- Personal attention level: 1-on-1, small group, or one-to-many
- Effort expectation: DIY (do it yourself), DWY (do with you), DFY (done for you)
- Delivery medium: in-person, phone, email, text, Zoom, chat
- Consumption method: audio, video, or written
- Speed of response: 24/7, 9-5, 5-min, 1-hour, 24-hour
- 10x test: if they paid 10x price, what would you provide? 1/10th price?
