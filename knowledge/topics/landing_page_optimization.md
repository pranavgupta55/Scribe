# Landing Page Optimization

> Dedicated ad landing pages eliminate multi-step friction and nearly double conversion vs. homepage.

## How-To / Method
- Build separate landing page from homepage, optimized only for ad traffic
- Eliminate navigation, logo prominence, and secondary CTAs
- Single headline: 'Free 18-Point Inspection' (specific, scannable)
- Include: phone number, form, locations, FAQs; remove everything else

## Key Numbers
- Each step in funnel loses ~50% of traffic; moving from homepage (multi-step) to dedicated page recovers lost traffic
- Potential 2x+ improvement from funnel optimization alone

## Caveats
- Redirecting all site buttons to dedicated landing page can improve existing organic site traffic conversion
- Dedicated page doesn't have to be navigable from main site
