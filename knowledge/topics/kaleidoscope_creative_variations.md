# Kaleidoscope Creative Variations

> Take one winning image and generate 20-30 variations via filters, AI, and remakes to extend reach.

## Framework
- AI filters: black and white, sepia, cartoonized, Ghibli style
- Generate 3-second video versions from static images
- Re-film same content with different production styles
- Test all variations while keeping copy unchanged

## Why It Works
- Audiences see fatigue differently by image format/style
- Same audience, different visual presentation = fresh campaigns
- Low effort to expand inventory; high ROI on creative testing
