# On-Target Earnings Reverse Engineering

> Set profit-share % by working backward from desired salary, not vice versa.

## How-To / Method
- Determine what ops manager should earn ($150K, etc.)
- Divide by profit to find % that yields target pay
- Example: $50K/month profit, $150K annual target = 10% profit share

## Key Numbers
- $50K monthly profit per store ÷ 12 months ÷ $150K target salary = ~10% profit share

## Key Takeaway
- If bad: fired; if mediocre: makes OTE; if crushes: makes higher; profit share adjusts automatically
