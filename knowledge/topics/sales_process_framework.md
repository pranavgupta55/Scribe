# Sales Process Framework

> Three-phase script: intro frame → discovery → match pain to solutions → pitch → sale questions only.

## How-To
- Intro frame: establish context without product talk
- Discovery: ask questions; let prospect state pain points
- Match: explicitly link each pain they mentioned to your solution set
- Pitch: here's how we solve each of these three problems you said
- Objection handling: answer sale questions (price, timeline), NOT product education

## Common Mistakes
- Script too long; founder explains all product details
- Founder closes and becomes the hero instead of prospect
- New seller hired and expected to replicate founder; gets discouraged when they can't
