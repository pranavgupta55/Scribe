# Operational Excellence

> Excellence requires identifying every constraint to success and solving for it systematically.

## How-To / Method
- Map all blocks to success: power, internet, inventory, logistics
- Don't assume one solution works; have backup for each
- All constraints + skilled execution = outcome

## Framework
- Work backwards from goal through all necessary conditions
- Treat each condition as solvable problem not fate
- Willingness to pay cost for each solution determines success
