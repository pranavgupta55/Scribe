# The Decision to Go Pro

> Going pro is a binary decision, not a gradual shift

## How-To / Method
- Ask: If I could do 100x more, would I hit my goals?
- If yes: What resources required? Is it worth it?
- If the answer to cost-benefit is yes: Decide. You're going pro.
